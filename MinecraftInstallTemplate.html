Installation Steps:

  1. Purchase Minecraft Java Edition: Ensure you have purchased and downloaded the Java Edition of Minecraft. This is crucial for our curriculum. You can get it from the official Minecraft website: minecraft.net/buy
  2. Install Java: Our courses require a specific version of Java. Please download and install Java 8 (JRE 8) from java.com/download.
  3. Install Minecraft Launcher: Download and install the Minecraft Launcher from the official website if you haven't already.
  4. Log in to the Launcher: Open the Minecraft Launcher and log in with your Mojang or Microsoft account.
  5. Select "Java Edition": Ensure you have "Minecraft: Java Edition" selected in the launcher.
  6. Create a new "Installation":
    • Go to the "Installations" tab.
    • Click "New Installation".
    • Name it "JetLearn".
    • For "VERSION", select "release 1.16.5" (or the version specified by your teacher).
    • Click "Create".
  7. Download and Install Forge (Optional, but Recommended for Mods):
    • Go to the Forge website: files.minecraftforge.net
    • Select the same Minecraft version (e.g., 1.16.5).
    • Download the "Installer" version.
    • Run the downloaded Forge installer and select "Install client".
    • Once installed, open the Minecraft Launcher, and you should see a "Forge" profile available in the dropdown next to the "Play" button. Select it.
  8. Test Your Installation: Launch the game using the "JetLearn" or "Forge" profile to ensure it loads correctly.

Minimum System Requirements:

  • Operating System: Windows 10/11, macOS, Linux
  • Processor: Intel Core i3-3210 3.2 GHz / AMD A8-7600 APU 3.1 GHz or equivalent
  • RAM: 4GB RAM
  • Graphics: Integrated: Intel HD Graphics 4000 / AMD Radeon R5 series. Dedicated: Nvidia GeForce 400 Series / AMD Radeon HD 7000 Series or equivalent
  • Storage: 1GB for game core, maps, and other files
  • Internet Connection: Stable broadband connection (at least 20 Mbps recommended)

If you encounter any issues during installation, please do not hesitate to contact our support team at support@jet-learn.com. We are here to help!
